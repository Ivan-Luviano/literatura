# 📚 Literalura

Literalura es una aplicación de consola desarrollada en **Java con Spring Boot** que permite buscar libros usando la API pública **Gutendex** y almacenar la información en una base de datos **PostgreSQL**.

El proyecto forma parte del challenge de **Alura + Oracle Next Education (ONE)**.

---

# 🚀 Funcionalidades

La aplicación permite:

1️⃣ Buscar libros por título utilizando la API Gutendex  
2️⃣ Guardar libros y autores en una base de datos PostgreSQL  
3️⃣ Listar todos los libros registrados  
4️⃣ Listar todos los autores registrados  
5️⃣ Buscar autores que estaban vivos en un año específico  
6️⃣ Listar libros por idioma  

---

# 🧰 Tecnologías utilizadas

- Java 21+
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- API Gutendex
- Jackson (para conversión de JSON)
